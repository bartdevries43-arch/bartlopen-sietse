# bartlopen — Run Coach (Sietse)

Persoonlijk halve-marathonschema (Op naar 21,1K — richting 1:45) voor Sietse, met coaching, stats en badges.
16 weken, maandag + zaterdag, **op tijd & gevoel gestuurd** (km zijn indicatief) en met knie-opbouw als rode draad.
Statische web-app — werkt op telefoon, laptop en iPad, en is te "installeren" als app.
